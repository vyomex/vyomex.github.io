Overview

The CAN 2.0B + AXI4-Lite Core is a Verilog IP providing a lightweight, synthesizable Controller Area Network (CAN 2.0B) controller wrapped with an AXI4-Lite bus interface. It enables seamless integration into FPGA/SoC platforms, offering standard CAN communication with configurable bit timing, transmit/receive buffers, and protocol-level handling.

Applications

  • Automotive networks
  • Industrial automation
  • IoT devices with CAN support
  • Embedded controllers & FPGA-based gateways
  • Test and measurement instruments

Key Features

  • CAN 2.0B compliant transmitter and receiver
  • AXI4-Lite wrapper for SoC integration
  • 64-byte address space, 32-bit data bus
  • Configurable bit timing (BRP, SJW, TSEG1, TSEG2)
  • Support for Standard (11-bit) and Extended (29-bit) identifiers
  • Remote and data frame support (RTR, IDE)
  • Transmit buffer with up to 64-bit payload
  • Receive buffer with data valid flag
  • Fully synthesizable RTL, simulation-ready

Architecture

The core consists of two primary layers:

  • CAN Controller Core – Handles protocol logic, bit timing, transmit/receive operations.
  • AXI4-Lite Wrapper – Provides host access to registers for configuration, transmit, and receive control.
