Frequently Asked Questions

Does AltDB work with my existing Altium DbLib setup?

Yes. AltDB uses a Microsoft Access (.accdb) database which connects directly to Altium Designer via the standard DbLib configuration. If you already have a DbLib set up, AltDB can manage the same database.

Do I need to install anything else on my machine?

AltDB is a standalone Windows desktop application. It requires the Microsoft Access Database Engine (free from Microsoft) and the 32-bit ODBC driver for Access. No Python or other runtime needed — everything is packaged in the installer.

Can multiple engineers use AltDB at the same time?

Yes — with the Pro or Enterprise license the database can be hosted on a shared network drive. Multiple users connect simultaneously with their individual credentials and roles.

What happens to our existing component data?

If you have existing data in Excel or another format we offer a Library Migration Service. We handle the full migration and verify every record before delivery.

Is our component data stored in the cloud?

No. AltDB is entirely local. Your database file lives on your machine or company network drive. No data is ever sent to external servers. Full data ownership stays with you.

What operating systems are supported?

Currently Windows only — which aligns with Altium Designer's primary platform. Compatible with all modern Windows versions (Windows 10 and 11).

Can we request custom features for our workflow?

Yes — Enterprise customers get custom feature development as part of the plan. Solo and Pro customers can submit feature requests which are considered for upcoming releases.
